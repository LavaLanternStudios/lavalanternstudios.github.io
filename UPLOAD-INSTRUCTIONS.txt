FINAL DIRECT YOUTUBE HERO FIX

This package uses ONLY the YouTube links already present in the website.
It does not require MP4 files and does not use the YouTube JavaScript API.

WHY THE PREVIOUS VERSION WAS BLACK

The previous API version hid the iframe until YouTube reported that playback
had started. If the API script, autoplay, an extension, or the video failed,
the iframe remained invisible and the hero stayed black.

THIS VERSION

- creates a normal YouTube iframe directly;
- never hides the iframe while it loads;
- sends an origin referrer to YouTube;
- requests muted autoplay;
- requests single-video looping;
- hides controls;
- disables keyboard and pointer interaction;
- starts one second into the video;
- enlarges and crops the player to move most YouTube edge chrome outside
  the visible hero;
- uses all YouTube links already stored in the existing content/data files.

INSTALLATION

1. Delete or ignore the earlier YouTube API and MP4 fix packages.
2. Extract this ZIP.
3. Copy its contents into the ROOT of the cloned repository.
4. Replace these seven files:

   style.css

   assets/js/home-content.js
   assets/js/games-content.js
   assets/js/fab-content.js
   assets/js/portfolio-content.js

   portfoliopages/portfoliostyle.css
   portfoliopages/assets/data/portfoliopage-loader.js

5. In GitHub Desktop, commit with:

   Restore direct YouTube hero videos

6. Push origin.
7. Wait for the Pages deployment to turn green.
8. Test in a Chrome Incognito window with extensions disabled.
9. Also press Ctrl + F5 on the normal browser window.

IMPORTANT YOUTUBE LIMITATION

YouTube controls can be hidden with controls=0. However, YouTube decides when
its own branding treatment appears. Its modestbranding parameter is deprecated
and no longer has any effect. Cropping the iframe removes most edge chrome, but
no website code can guarantee that YouTube will never display any branding or
error overlay.

DIRECT EMBED TEST

For each main-page video, open its embed link directly in an Incognito window.
For example:

https://www.youtube.com/embed/2oTJp0fKoIc

If that direct YouTube page says “This video is unavailable”, the website code
cannot play that particular ID. The link, visibility, embedding permission, or
YouTube account/video restriction must be corrected for that video.
