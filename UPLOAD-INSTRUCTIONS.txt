INSTALLATION

1. Extract the ZIP.
2. Copy its contents into the root of the cloned repository.
3. Allow all six matching files to be replaced.
4. Do NOT copy anything from the earlier MP4 package.
5. Open GitHub Desktop.
6. Commit with:
   Use YouTube API for hero videos
7. Push origin.
8. Wait for GitHub Pages to deploy.
9. Open the site in an Incognito window or press Ctrl + F5.

BEHAVIOUR

All main-page and portfolio-detail YouTube hero videos:

- use the existing YouTube links;
- start muted;
- request autoplay through the official YouTube IFrame Player API;
- loop continuously;
- restart if YouTube reports the video has ended;
- retry after returning to the tab;
- retry after a browser autoplay block on the first interaction;
- hide controls;
- ignore pointer input;
- crop to cover the hero area.

Normal YouTube videos inside the body of portfolio detail pages remain
interactive and are not changed.

DIAGNOSTIC INFORMATION

If YouTube still refuses one particular video, open the browser console.
The script will report YouTube's numeric player error code with the video ID.
That distinguishes a code problem from a video-level embedding restriction.
